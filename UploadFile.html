Pengaturan

1

Buat Akun & Project

Buka supabase.com dan buat project baru.

2

Buat Bucket "images"

Di menu Storage, buat bucket baru bernama images. PENTING: Centang "Public bucket".

3

Atur Policies

Di tab Policies, buat policy baru (INSERT, SELECT, DELETE) untuk role 'anon' dan 'authenticated'.

Copy URL & Key

Ambil URL dan Public Anon Key di Project Settings -> API.
