üìñ Cara Penggunaan

Untuk Mengenkripsi:

  1. Pilih tab Gambar atau Video
  2. Upload satu atau lebih file
  3. Masukkan kunci enkripsi (ingat kunci ini!)
  4. Klik "Enkripsi Semua" atau enkripsi per file
  5. Download hasil atau download semua sebagai ZIP

Untuk Mendekripsi:

  1. Upload file yang sudah terenkripsi
  2. Masukkan kunci yang sama saat enkripsi
  3. Klik "Dekripsi Semua" atau dekripsi per file
  4. File asli akan muncul kembali

üîê Metode Enkripsi:

  • Gambar: XOR dengan pixel shuffling menggunakan PRNG seeded key
  • Video: ChaCha20-like stream cipher pada byte data mentah
